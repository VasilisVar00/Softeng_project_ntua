# NodeJs Packages

```
"dependencies": {
    "axios": "^0.24.0",
    "commander": "^8.3.0",
    "inquirer": "^8.2.0",
    "listener": "^1.0.1"
  }
  ```
  
  Εγκατάσταση :
  
  ```
  npm init  (για να δημιουργήσουμε ένα νέο npm package) 
  
  npm --save install inquirer listener axios commander
  
  npm link
  
 ```
 
 - Για τα child processes : execSync

Εγκατάσταση: ```npm install execSync```
